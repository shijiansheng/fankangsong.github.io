以下是我用到的vim快捷键和命令，后续累积再添加。2014年12月29日更新。

##视图

`i` 编辑视图

`esc` 阅读视图
`v` 复制视图

##光标移动

`hjkl` 左下上右

`b` 光标往后一个词

`e` 光标向前一个词

`0` 光标至行首

`gg` 屏首

`GG` 屏末

`ctrl + f` 下翻一屏

`ctrl + b` 上翻一屏

`ctrl + u` 上翻半屏

`ctrl + d` 下翻半屏

`ctrl + o` 光标跳至前一次编辑的地方

`ctrl + i` 光标跳至上一次编辑的地方

##编辑

`a` 光标前输入

`A` 当前行末插入

`o` 光标后输入

`O` 在光标上一行插入

`:u` 撤销一次

##文件

`:w` 保存

`:q` 退出

`:q!` 不保存退出
